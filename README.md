# Invasion Ferrocerium
## About
Invasion Ferrocerium is a mod that adds alien robots that invade your world.
### Invasion Mechanic
Each dimension has an invasion level. The higher the level the more dangerous things get.
### Mobs
- Ant Scout Bot
- Ant Soldier Bot
- Alien Helicopter Bot
### Items
- Aluminium
- Alien Parts
  - Dropped by the Ant Scout Bot
  - There are three types
- Crude Alien Beacon
  - Used to progress the invasion
- Alien Beacon
  - Made using the alien parts
  - Used to reduce the progress of the invasion
- Strange Alloy
- Strange Alloy Tools
  - Better than diamond tools.
## Future Developments
The mod is expected to grow in content over the course of many updates increasing in number of mobs, items and other mechanics.
### Other Versions/Mod Loaders
There are no current plans to port this mod to other versions or mod loaders, however, this may change in the future.
